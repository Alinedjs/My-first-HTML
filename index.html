TAG HTML

This tag delimit the entire document of an html code (we use to open and close the text, only what is between them will be considered to configure the site)

TAG HEAD

Space used to place page header instructions this is the first thing it loads before displaying it to the user.

TAG TITLE

This will be the title of the site page (which appears at the top of the page in the browser)

TAG BODY

Space where all the graphic and visual content of the web page appears (what it will show the user when they access the site)
