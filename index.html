WHAT IS HTML

HTML stands for Hyper Text Markup Language and it is the language of the internet that defines the structure and content of websites and formats web page, it uses different tags and elements to identify what you see on screen.

TAG HTML

TAG BODY

(back)

Space where all the graphic and visual content of the web page appears (what it will show the user when they do not access the site).

Some elements contained in it are:

  1. h1> used to create titles.
  2. p> used to create paragraphs in text.
  3. a> used to define a hyperlink, which is used to link from one page to another.
  4. ul> used to create unordered lists.
  5. ol> used to create ordered lists.
